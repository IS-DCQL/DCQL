# Final-exam answer keys + grading rubric (§6.3)

Reference answers and grading points for the seven blank papers in `../exam-blank/`.
See `../exam-blank/README.md` for the full structure, category mapping, and authored items.

Each item provides:
- **Business intent** — the business intent of the task.
- **Reference answer** (writing items) — the standard implementation, taken verbatim from the §6.2
  `../../conciseness/` corpus (or an authored reference for the two non-corpus items).
- **Expected result** (prediction items) — the concrete returned records on the teaching dataset
  shipped under `../exam-blank/dataset/`.
- **Statement shown on the paper** (reading/prediction items) — the code shown on the paper, for the
  grader to check against.
- **Grading notes** — graded with the §6.3 four error classes: **S** syntax-structure, **L**
  semantic-logic, **V** numeric, **M** minor-language. The writing/reading/prediction sub-scores
  total 70 / 15 / 15.

Regenerate with `../_generate_exams.py`.
